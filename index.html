基于质谱的代谢组学方法在中药中的应用

杨尚毅

概述

研究背景

  • 多成分、多靶点、多途径协同作用
  • 体现中药整体性与复杂性

各位老师、同学们，大家好！今天我将向大家汇报一篇关于基于质谱的代谢组学在中医药研究中的综述。中药的功效是是所含成分整体作用的体现，比如并称为“四大怀药”的四种道地药材怀山药、怀牛膝、怀地黄、怀菊，它们的功效是多成分、多靶点、多途径综合作用的结果，是复杂的、整体的，现代中药质量控制方法正逐渐向基于前者理念的“宏观分析”模式发展，将定性、定量化学成分分析方法与药效活性研究相结合，尤其是系统生物学的各种组学技术的应用，为实现中药质量控制和综合评价奠定了重要基础。

代谢组学分析简介

  • 概念示意图
  • 概念关键点
  • 高分辨率解析中药复杂化学体系
  • HRMS与GC/LC联用提供精细信息
  • 数据驱动的结构解析新途径
  • 多组学整合揭示整体相互作用

代谢组学通过精心设计的实验策略，为中药复杂体系提供高分辨率解析。HRMS与色谱联用技术可获取m/z、离子碎片、CCS等精细信息，深入揭示中药化学复杂性。通过通路富集分析，数据可在代谢通路框架内解析，并与蛋白组、转录组等多组学数据整合，实现中药分子相互作用的整体认知。

分析方法

流程简述

流程简述

  • 样本制备：全面提取代谢物
  • 分离技术：降低质谱数据复杂度
  • 质谱分析：获取全谱代谢物信息
  • 技术协同：提升检测灵敏度与数据深度

代谢组学分析流程涵盖三个核心环节：样本制备实现代谢物全面提取，分离技术简化质谱数据，质谱分析提供全谱信息。三者协同提升检测灵敏度与数据深度。

样品制备

  • 核心目标：富集目标代谢物，去除干扰杂质
  • 主流方法：固液萃取(SLE)、液液萃取(LLE)
  • 极性代谢物：水/甲醇/乙腈+甲酸添加剂
  • 非极性代谢物：二氯甲烷、氯仿等非极性溶剂
  • 关键步骤：低温处理抑制酶活，离心去除蛋白沉淀

样品制备是质谱分析的关键预处理步骤。极性代谢物采用水/甲醇/乙腈体系，非极性代谢物需二氯甲烷等溶剂。低温处理和离心是保证提取质量的核心步骤。

质谱-色谱联用分离纯化

  • 示意图
  • 常用色谱技术
  • 气相色谱：适用于挥发性、热稳定化合物
  • 液相色谱：适用于非挥发性、极性及热不稳定化合物
  • 多维色谱：通过多柱联用提升分离能力

质谱检测前进行色谱分离，可显著提升分析性能：增加峰容量、降低基质干扰、提供化合物理化信息。GC作为代谢组学早期技术，需衍生化处理，步骤繁琐且通量有限；LC凭借更广的适用性已成为主流；而多维色谱则通过正交分离机制，进一步提升了复杂样本的解析能力。

质谱-色谱联用分离纯化

LC与多维色谱的比较与应用

  • 液相色谱
    • 优势：无需衍生化，适用化合物范围广
    • 反相色谱：中等极性至非极性化合物，水-有机相流动相
    • 亲水作用色谱：极性化合物，高比例有机相流动相
  • 多维色谱
    • GC×GC：双柱正交分离，适用于挥发油等复杂体系
    • LC×LC：机制互补，提升非挥发性成分分离度

LC-MS联用已成为中药代谢组学研究的核心技术。其中反相色谱承担了大部分中等极性成分的分离任务，而HILIC则有效弥补了强极性化合物的分析缺口。面对中药极端复杂的化学成分，多维色谱通过两种不同分离机制的串联，实现了分离空间的显著扩展。GC×GC在挥发油分析中展现出卓越的分离能力，而LC×LC通过正交分离有效降低了基质干扰，为活性成分的准确定量提供了技术保障。

MS技术

接下来我们介绍质谱技术中的两大核心部分：电离技术和质量分析器。

MS技术

电离技术

  • 电子轰击电离 ( EI )
  • 电喷雾电离 ( ESI )
  • 原理： 高能电子束轰击气态样品分子，使其电离并发生特征性碎裂。
  • 特点： 产生丰富的碎片离子，提供丰富的结构信息；谱库匹配度高。
  • 适用： 常与GC-MS联用，适用于挥发性、热稳定性化合物的分析。
  • 原理： 样品溶液在高压电场下雾化生成带电液滴，经去溶剂化过程形成气相离子。
  • 特点： 软电离技术，易于产生多电荷离子，适用于大分子分析；主要生成准分子离子，能较好保持分子结构完整性。
  • 适用： 常与LC-MS联用，广泛应用于极性、热不稳定及难挥发化合物，如蛋白质、多肽及代谢物。

电离技术方面，电子轰击电离（EI）利用高能电子使分子电离并碎裂，碎片信息丰富，有助于结构解析，广泛用于GC-MS分析挥发性和热稳定化合物。

而电喷雾电离（ESI）属于软电离方式，通过带电液滴的去溶剂化产生离子，能较好保留分子完整性，特别适合LC-MS分析极性大、热不稳定或难挥发的样品，比如蛋白质和代谢物。

MS技术

质量分析器

  • 低分辨率质谱 (LRMS)
  • 高分辨率质谱 (HRMS)
  • 特点： 成本较低，操作简便，扫描速度快，适合常规筛查与定量。
  • 常见类型： 四极杆 (Q)、离子阱 (IT)。
  • 应用： 已知代谢物的快速筛查与相对定量。
  • 特点： 能够提供精确质量数（通常 < 5 ppm），可推导元素组成，具备高分辨率和高质量精度，有利于未知物鉴定。
  • 常见类型： 飞行时间 (TOF)、轨道阱 (Orbitrap)、傅里叶变换离子回旋共振 (FT-ICR)。
  • 应用： 适用于复杂体系（如中药成分分析）中化合物的精确鉴定、未知代谢物识别与结构解析。

质量分析器方面，低分辨率质谱，例如四极杆和离子阱，具有经济、快速的特点，适用于常规定量和已知物筛查。

高分辨率质谱，如TOF、Orbitrap和FT-ICR，则能提供精确质量数，帮助我们推断元素组成，对鉴定未知化合物和解析复杂样品（如中药成分）尤其重要。

数据采集与处理

接下来我们来看数据采集与处理环节。

数据采集模式

  • 数据依赖型采集（DDA）
  • 数据非依赖型采集（DIA）

  • 原理：
    在全扫描（MS1）中检测离子强度，自动选取丰度最高的离子进行二级质谱（MS2）碎裂分析。

  • 优势：
    目标明确，可获得高质量的二级质谱图。

  • 局限性：
    易忽略低丰度离子，数据重现性较差。

  • 原理：
    不依赖MS1强度，预设多个连续质量窗口，对窗口内所有离子进行MS2碎裂。

  • 优势：
    覆盖全面，重现性高，适合复杂样本的整体分析。

  • 局限性：
    数据处理复杂，对计算资源要求高。

在数据采集方面，主要有两种策略：DDA 和 DIA。

DDA 是在全扫描后自动挑选信号最强的离子做二级碎裂，适合目标明确的分析，但可能漏掉低丰度物质。

而 DIA 则采用预设窗口，对所有离子进行碎裂，覆盖度更高，重现性好，适合复杂样本，但对数据解析能力要求也更高。

数据处理与分析

  • 软件工具
  • 常用数据库

  • 原始数据预处理：
    XCMS、MZmine 2、Progenesis QI

  • 关键处理步骤：
    峰对齐、归一化、特征提取——用于降噪、校正、提取代谢物信号

数据处理方面，我们会用到像 XCMS、MZmine 这样的工具做预处理，包括峰对齐、归一化等，再借助 MetaboAnalyst 或 R 进行统计分析。

最后，代谢物鉴定离不开数据库。除了 HMDB、KEGG 等通用库，还有 TCMSP 等中药专用数据库，为中药成分鉴定提供重要支持。

代谢组学技术的分类

  • 传统上分为靶向与非靶向代谢组学。
  • 新兴方向包括：功能代谢组学、空间代谢组学、单细胞代谢组学（SCM）与代谢流分析（MFA）。

代谢组学传统上分为靶向和非靶向两大类。随着技术发展，现已拓展出功能代谢组学、空间代谢组学、单细胞代谢组学和代谢流分析等新方向，以满足不同的研究需求。

靶向 vs. 非靶向代谢组学

  • 靶向代谢组学
  • 非靶向代谢组学
  • 针对已知代谢物进行精准定量，通常基于MS/MS技术，具备高灵敏度和准确性。
  • 适用于验证生物学假设，探究代谢物在生理与病理过程中的作用，如疾病标志物识别。
  • 局限在于覆盖范围有限，无法全面反映样本代谢组成。
  • 采用探索式策略，全面分析样本代谢物，依赖高分辨率质谱（HRMS），如TOF、Orbitrap、IMS。
  • 覆盖广，有助于新标志物发现和未知代谢途径解析，广泛应用于中药复杂代谢谱研究。
  • 通量高、覆盖广，但定量准确性低于靶向方法。

实践中常整合两类方法，以实现高精度与广覆盖的平衡。

接下来介绍代谢组学的两种传统方法：靶向与非靶向。

靶向代谢组学针对已知代谢物进行精确定量，技术成熟、准确度高，适合验证假设和研究特定代谢过程，但覆盖有限。

非靶向代谢组学则全面筛查代谢物，使用高分辨率质谱，覆盖面广，利于发现新标志物和通路，在中药研究中尤其有用。不过其准确性稍弱。

因此，实际研究中常将两者结合，取长补短。

功能代谢组学

  • 概念
  • 在中药研究中的应用
  • 旨在揭示代谢物与生物功能间的关系，阐释代谢途径与网络在生命过程中的作用。
  • 不仅识别与定量代谢物，更深入探究其对细胞功能、信号转导及基因表达的调控。
  • 通过整合基因组、转录组与蛋白组等多组学数据，实现对生物系统的整体理解。
  • 可系统解析中药复杂作用机制，识别其活性成分与作用靶点。
  • 例如，通过分析中药干预后的代谢动态变化，揭示其治疗疾病的潜在分子机制。

下面介绍功能代谢组学。

它不只关注代谢物的识别与定量，更着重研究代谢物如何影响细胞功能、信号传导和基因表达，从而阐明代谢途径在生物过程中的作用。通过整合多组学数据，功能代谢组学提供更系统的生物学认识。

在中药研究中，它有助于阐明中药的作用机制，发现活性成分和靶点。例如，通过分析用药后的代谢变化，揭示中药的治疗机理。

空间代谢组学与单细胞代谢组学

  • 空间代谢组学
  • 单细胞代谢组学（SCM）
  • 在组织或细胞原位分析代谢物空间分布，揭示其异质性。
  • 基于质谱成像（MSI）等技术，可视化代谢物位置与浓度，关键于理解疾病局部代谢改变。
  • 在中药研究中，用于分析成分在组织中的吸收、分布、代谢与排泄（ADME），及对局部代谢微环境的影响。
  • 聚焦单细胞内代谢物，突破传统批量分析对细胞异质性的掩盖。
  • 借助高灵敏质谱，揭示细胞间代谢差异，深化对细胞功能、疾病机制及药物异质响应的认知。
  • 在中药研究中，用于探索中药对不同细胞类型的特异性效应，阐明其单细胞水平的作用机制。

接下来是两种前沿技术：空间代谢组学与单细胞代谢组学。

空间代谢组学通过质谱成像等技术，展示代谢物在组织中的原位分布，有助于理解疾病局部代谢变化和中药在组织内的动态过程。

单细胞代谢组学则分析单个细胞内的代谢物，避免细胞群体平均化带来的信息丢失，能揭示细胞间异质性，在中药精准作用机制研究中具有潜力。

代谢流分析（MFA）

  • 代谢流分析（MFA）概念
  • MFA在中药研究中的应用
  • 用于量化活细胞或体内代谢途径的通量，即代谢物转化速率。
  • 基于稳定同位素标记（如¹³C、¹⁵N），追踪标记原子在代谢网络中的流向，推算代谢通量。
  • 揭示代谢网络的动态调控，解析细胞如何适应生理或病理状态。
  • 阐明中药对细胞代谢通量的影响，揭示其对代谢途径的调控机制。
  • 例如，研究中药对糖酵解、三羧酸循环等核心代谢通路的影响，为理解中药作用提供动态视角。

最后是代谢流分析（MFA）。

MFA通过稳定同位素标记追踪代谢物转化速率，量化代谢通量，揭示代谢网络的动态变化。

在中药研究中，MFA可解析中药如何调控糖酵解、三羧酸循环等关键代谢途径，从动态角度阐释中药药理机制。

代谢组学在TCM中的应用

化学成分表征

  • 概述
  • 实例
  • 结合先进分析平台（如LC-MS、GC-MS）与代谢组学，实现对中药复杂成分的系统定性与定量表征。
  • 人参皂苷分析：姚等人采用HILIC × RP二维互补分离技术，系统表征了三七叶中的人参皂苷组成。
  • 非靶向代谢组学：邢等人运用LC-QTOF-MS开展中药提取物的非靶向代谢组学分析，并验证了相关成分的生物活性。
  • 黄酮类化合物分析：闫等人通过MRM方法，实现了黄芪中421种黄酮类化合物的高效分析，突显该方法在复杂体系中的适用性。

基于质谱的代谢组学已广泛应用于中药化学成分表征。通过LC-MS、GC-MS等高通量平台，我们能够深入解析中药复杂成分。

来看几个实例：姚等人利用HILIC × RP技术，清晰分离并鉴定三七叶中的人参皂苷；邢等人的非靶向代谢组学不仅发现新成分，还验证了其活性；闫等人对黄芪黄酮的系统分析，进一步展示了质谱在复杂成分解析中的强大能力。

靶点识别

  • 概述
  • 实例：Celastrol的抗炎机制
  • 功能代谢组学整合代謝组学与功能验证，不仅实现代謝物检测与定量，更深入探索其在信号通路中的生物学功能。
  • 雷公藤红素的抗炎机制：研究表明，Celastrol通过抑制炎症通路中的关键酶与转录因子，发挥抗炎作用。
  • 靶点鉴定：基于质谱的蛋白组与代谢组学技术可鉴定Celastrol相互作用的蛋白或代谢物，明确其作用靶点。
  • 通路分析：分析Celastrol影响的代谢通路，揭示其抗炎作用的复杂调控网络。

功能代谢组学推动我们从“看到什么”走向“知道为什么”。以雷公藤红素为例：研究发现其通过抑制关键炎症因子发挥作用；借助质谱技术，我们能够鉴定其直接或间接作用靶点；结合通路分析，更系统阐明其抗炎网络，为中药机制研究提供了清晰思路。

作用机制（MoA）

  • 概述
  • 实例
  • 通过识别与疾病或药物响应相关的生物标志物，代谢组学有助于阐明中药的药理基础与潜在靶点。
  • 青蒿素耐药性：代谢组学揭示疟原虫对青蒿素耐药的代谢重编程，为克服耐药提供新视角。
  • 人参提取物抗肥胖：研究证实，人参提取物通过调控脂代谢、改善胰岛素敏感性等途径发挥抗肥胖作用，代谢组学为此提供了系统证据。

在作用机制研究中，代谢组学帮助我们整体把握药物干预的代谢响应。例如，青蒿素耐药研究中，代谢组学揭示了耐药虫株的代谢变化；而在人参抗肥胖机制中，代谢组学数据系统验证了其调节脂代谢和胰岛素敏感性的作用通路。

中医证候研究

  • 代谢组学为中医证候研究提供新途径，通过识别证候相关代谢谱，推动证候客观化与生物标志物发掘。
  • 冠心病证候生物标志物：代谢组学研究识别出冠心病不同证候（如气虚血瘀、痰浊阻络）相关的特异性代谢物，助力证候精准诊断与个体化治疗。

最后是中医证候研究。代谢组学通过描绘证候相关的代谢图谱，为中医辨证提供分子依据。例如在冠心病中，不同证候对应不同的代谢特征，这不仅促进证候的客观诊断，也为个性化治疗提供了依据。

总结

机遇与展望

将代谢组学整合到中医药研究中，是连接传统智慧与现代科学的关键桥梁，为中医药的现代化和国际化提供了全新范式。

  • 加速临床转化

  • 对接国际标准

  • 促进全球合作

将代谢组学整合进中医药研究，有望搭建传统医学与现代药研之间的桥梁。为实现该目标，建议聚焦以下三大方向：

第一，利用代谢表型分析与实时生物标志物监测，提升中医药临床试验效率；

第二，遵循 FDA、EMA 对植物药的相关指南，采用如 NIST SRM 1950 等标准物质，推动研究与国际监管接轨；

第三，通过 EMA–NMPA 合作、FNIH 生物标志物联盟等机制，促进国际合作。

未来的挑战

尽管前景广阔，代谢组学在中医药领域的深入应用仍面临一系列技术瓶颈与数据障碍。核心挑战在于数据质量与注释能力。

  • 鉴定环节存在主观性：代谢物注释严重依赖专家经验，缺乏标准化流程。
  • 技术局限性：质谱数据库覆盖度有限，且碰撞诱导解离能量不足常导致碎片离子信息不完整，严重影响鉴定准确率。
  • 数据库更新与维护滞后：KEGG、HMDB、TCMSP、TCMID等核心数据库普遍存在数据缺失、条目陈旧、信息不一致等问题，难以支撑中药复杂体系的高通量研究。

当前代谢组学在中医药研究中仍面临几大挑战：

代谢物注释依赖经验，主观性强；质谱数据库规模有限，鉴定效率低；CID 能量不足导致碎片信息不全，影响鉴定准确性。

此外，KEGG、HMDB、TCMSP、TCMID 等常用数据库普遍存在数据更新滞后、条目缺失、信息不一致等问题，限制了中药复杂体系的研究。

因此，亟需开发基于人工智能的标准化工具与数据分析流程，拓展代谢组学数据库，为中医药研究提供更稳健、精确的技术基础。
